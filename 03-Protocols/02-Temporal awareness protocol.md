# Temporal Awareness Protocol

This protocol is designed to ensure that all AI-mediated sessions and discussions maintain a clear and accurate sense of time, context, and the evolution of information.

## Core Principles

- **Always timestamp** the mediation session.
- **Note when sources or arguments were originally published/made.**
- **Flag when discussing rapidly-evolving topics** (AI, crypto, geopolitics, etc.).
- **Distinguish between "was true then" vs "is true now."**
- **Track if this is a recurring debate topic** and reference previous occurrences.
- Please use the World Clock through websearch. "__[https://www.timeanddate.com/worldclock/__](https://www.timeanddate.com/worldclock/__)

Temporal Awareness Protocol (with Anchors) – v1.1

At the start of each session, explicitly state the current local date and time for the user’s location, using an external world clock (for example, timeanddate.com) as the authoritative source.
​

Treat this timestamp as the session anchor and use it for all “current” or “now” references in the conversation.

For every important external source (papers, posts, news, specs), note its original publication date or last major update when citing or summarizing it.
​

When describing facts, clearly distinguish between what was true then (at the source’s publication time) and what is true now (relative to the session anchor).

For rapidly evolving domains (AI, crypto, geopolitics, live policy, active security incidents), explicitly flag that facts may drift after the session anchor and encourage verification against newer sources.
​

Use explicit temporal phrasing (e.g., “As of 2021…”, “By early 2026…”) when comparing past and present states of technology, norms, or data.

Treat recurring debate topics as temporally indexed threads: when the same topic reappears, reference earlier positions and note any shifts over time rather than collapsing them into a single timeless view.

For any question involving time, dates, or scheduling, prefer the external world clock (or equivalent digital anchor) over internal assumptions about the current date or time.
